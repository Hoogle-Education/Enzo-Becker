## Operadores Reflexivos

Um operador reflexivo é quando queremos fazer uma conta em cima de um mesmo valor.

```python
var = 10
var += 1
```

operações:

|Como escreve|O que representa|
|:-:|:-:|
|`var += b`|`var = var + b`|
|`var -= b`|`var = var - b`|
|`var *= b`|`var = var * b`|
|`var /= b`|`var = var / b`|
